# Music Player

## Description
It is a UI for music player including features like:
  - Spinning images
  - Song deatils popup
  - Play and pause functionality
  - Switch songs
  - Progress Bar
  - Neon effect

## Stacks Used
* HTML  
* CSS
* JavaScript
